## 2 ways to shut down a Hyperbeam computer

### DELETE /v0/vm endpoint

You can manually delete a computer by hitting the `DELETE /v0/vm` endpoint.

<DeleteComputer />

### Configuring timeout values

The easiest way to manage the computer lifecycle is by configuring the computer's timeout values.

For a full example of timeout

#### Default timeout values

If no timeout configuration is specified, the following configuration will be applied:

```json
{
  "timeout": {
    "offline": 3600,
    "warning": 60
  }
}
```
